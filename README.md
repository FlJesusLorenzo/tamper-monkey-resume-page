Script para resumir la pagina en la que te encuentras

Pulsar para instalar: https://github.com/FlJesusLorenzo/resume-page/raw/refs/heads/main/main/script.user.js

Utiliza la clave de API de Google que podeis sacar desde aquí: https://aistudio.google.com/api-keys

Una vez teneis la clave, instalais el script y cuando recargueis una página os pedirá esa misma clave, solo la pedirá una vez
<img width="530" height="301" alt="image" src="https://github.com/user-attachments/assets/27a34501-a901-4cf3-9e3b-bc84d129ca53" />

La agregais y guardais, tendreis en la página un botón en la zona inferior derecha de la pantalla

cuando lo pulseis aparecerá una alerta donde podreis escribir lo que necesitais resumir o analizar ed la pagina en la que os encontreis

y al enviar aparecerá un Popup con el resumen/contestación a vuestra respuesta, este lo podreis copiar o cerrar
